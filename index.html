Anime Explorer

Descripción: Anime Explorer es una aplicación web que permite consultar animes populares y buscar por nombre. Utilizando la API de Jikan, los usuarios pueden explorar animes más vistos, aquellos que están en emisión en tiempo real y filtrarlos por género. Curriculum Editor

Descripción: Curriculum Editor es una aplicación web diseñada para ayudar a los usuarios a crear su currículum vitae (CV) personalizado. HTML, CSS, JavaScript, Tailwind CSS

Backend:
