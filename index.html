Aplicación de Inventario

Descripción: Esta aplicación de inventario permite a los usuarios gestionar el stock de productos de manera eficiente. Utiliza **MySQL** como base de datos para almacenar toda la información relacionada con los productos, categorías, y movimientos de inventario.
