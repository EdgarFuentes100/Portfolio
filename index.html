Habilidades Técnicas

Desarrollo Web

He desarrollado sitios web utilizando React.js, HTML5, CSS3, Bootstrap y JavaScript. Me enfoco en crear interfaces que sean funcionales y fáciles de navegar.

Desarrollo Móvil

He trabajado en la creación de aplicaciones móviles usando Android Studio, con Java y XML. Mi objetivo es crear aplicaciones móviles con un buen rendimiento y una interfaz intuitiva.

Desarrollo de Aplicaciones de Escritorio

He desarrollado aplicaciones de escritorio con C# y Windows Forms en Visual Studio, orientadas a mejorar la productividad y automatizar tareas en entornos de trabajo.

Desarrollo de APIs

He trabajado en la creación de APIs utilizando ASP.NET con C# en Visual Studio y con Node.js y Express en Visual Studio Code, para facilitar la comunicación entre diferentes aplicaciones.

Gestión de Bases de Datos

He manejado bases de datos en MySQL y SQL Server, creando y gestionando estructuras de datos que permiten almacenar y organizar la información de manera eficiente.

Autenticación y Seguridad

He implementado sistemas de autenticación utilizando OAuth2 con Google y Microsoft, para garantizar que solo usuarios autorizados accedan a las aplicaciones.

Desarrollo en Excel (VBA)

He creado aplicaciones personalizadas en Excel mediante Visual Basic for Applications (VBA), desarrollando formularios interactivos y automatizando procesos dentro de hojas de cálculo.
